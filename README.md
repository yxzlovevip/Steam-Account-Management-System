## 🚀 快速开始

### 第一步：启动程序

### 第二步：添加账号
1. 点击"➕ 添加账号"
2. 输入Steam账号和密码
3. 点击确定

### 第三步：登录账号
1. 在列表中找到账号
2. 点击"🚀 登录"按钮
3. 确认后Steam将自动启动

## ⚠️ 重要提示

### 使用前必读
1. **Steam安装**：确保已安装Steam客户端
2. **首次登录**：可能需要邮箱或手机验证码
3. **数据备份**：建议定期备份数据库文件
4. **密码安全**：密码只能在当前Windows用户下解密

### 数据存储位置
```
C:\Users\<你的用户名>\AppData\Local\SteamAccountManager\accounts.db
```

### 备份方法
复制上述数据库文件到安全位置即可
